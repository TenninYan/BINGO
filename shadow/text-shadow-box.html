Text Shadow

text shadow box, June 2009

Move your cursor over the box to cast a shadow with the spotlight. This demo uses the CSS property text-shadow, supported in Safari, Firefox 3.5, Opera, and Chrome.

Also check out my more sophisticated example that replaces the PNG spotlight with one generated by a CSS radial gradient and also casts a proper shadow for the ledge.
